Passport Requirements:

Evidence of Citizenship

  • State certified birth certificate. Hospital birth records are NOT accepted
    OR
  • Previous U.S. Passport, Original Naturalization or Citizenship Certificate
    (Children 0–13 years with previous passport, must also submit birth certificate)

Proof of Identity

  • State drivers license - If applicant is under age 16, parent's id must be used
    OR
  • Valid U.S. Passport OR Military ID OR State ID card only if accompanied by other forms of valid picture id (see passport agent for more information)
